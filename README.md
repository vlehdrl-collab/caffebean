<h3>프로젝트를 하면서 느낀점</h3>
한분은 이과정을 들으면 안되는분이셨고, 또 한분은 바이브코딩을 하는 분이라서 많이 힘들었던 것 같습니다. 어떻게든 이분들과 함께 최종프로젝트를 마무리 해야했기에
강사님에게 도움을 요청하였습니다. 골격은 짜는데는 gpt의 도움을 받았는데, 골격이 너무 엉성해서 각자가 주말동안 레이아웃을 만들고 월요일에
가장 나은것을 하기로 했습니다. 저혼자 책과 유튜브 강의를 들으면서 해왔고 아무도 해오지 않아서 기여율이 100%가 될 수 밖에는 없었습니다. 기간이 얼마남지 않았다는이유로
게시판을 바이브 코딩으로 만들어내었고 최종프로젝트의 발표를 마친 후, 로그인과 회원가입은 바이브 코딩으로 만들어내고 코드리딩을 하면서 저 혼자 따로 공부할 수 밖에는 없었습니다.
마지막 프로젝트인데 정석대로 배우지 못한것 같아서 많이 아쉬웠습니다. 부족한 부분은책을 보면서 메꿔나갈 생각입니다.
